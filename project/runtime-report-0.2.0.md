## 地图滚轮缩放修复（r6）

1. 地图世界尺寸不再除以 `camera.zoom`，避免与 `ctx.scale()` 相互抵消。
2. 滚轮使用指数缩放，并保持指针下方的世界坐标不漂移；范围限制为 `0.8–2.0`。
3. 地图容器使用稳定响应式高度，Canvas 不再通过 `height: 100% + min-height` 形成尺寸反馈。
4. `ResizeObserver` 仅在像素尺寸实际变化时重设 Canvas，避免重复清空与回调循环。
5. 运行时预览中的真实滚轮结果：缩放值 `1.000 → 1.616 → 1.162`；Canvas 始终为 `748×444`，页面 `scrollY` 始终为 `0`。
6. 完整页面截图确认底图、区域标记和访客标记均正常绘制；浏览器控制台无 error/warning。
