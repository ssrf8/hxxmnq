3. 确认进入主庭园后，页面显示填写的数据、春季第 1 日清晨、物资 6、灵感 1。

5. 在庭园中发送第一次真实行动，确认此时才发生第一次 LLM 生成。
6. 验证同一开场资料重复提交幂等，不同资料不能覆盖已经提交的开场。
